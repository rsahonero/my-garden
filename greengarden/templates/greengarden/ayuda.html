Condiciones Atmosfericas

El objetivo de este segmento es brindar al usuario la informacion de las condiciones atmosfericas.

Si se hace click sobre el boton "Actualzar", el mismo mostrara el tiempo en el que se realizo el ultimo escaneo, y cuanto falta para el siguiente. De a cuerdo a los datos mostrados el sistema experto calcula mediante un encadenaminto hacia adelante las posibles plagas y/o enfermedades que pueden estar afectando al cultivo de flores y rosas.

Inferencia

Mediante el uso de los datos obtenidos por la monitorizacion el sistema experto realiza un encadenamiento hacia atras para confirmar una de las posibilidades.

Al presionar el boton continuar el sistema experto buscara una regla asociada a una de las probabilidades, luego de hallarla si desconoce el valor, preguntara al usuario por el mismo.

Social

Green Garden es un proyecto de codigo abierto y cualquier contribucion es bien venida.

Cuestionario

Pregunta al usuario por el valor de un hecho en particlar, cabe mencionar que es obligatorio seleccionar una de las dos alternativas mostradas.

Explicacion

Si se tuviese alguna duda sobre un hecho en particular el sistema experto muestra una descripcion mas detallada del hecho en cuestion.
